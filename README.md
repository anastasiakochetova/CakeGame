# 🎂 CakeGame — учебный проект на C# WinForms

Оконная игра в стиле Purple Place. Игрок поэтапно собирает торт на конвейере, сравнивая результат с образцом на карточке.

## 📋 Этапы разработки

| Неделя | Этап | Статус |
|--------|------|--------|
| 1 | Окно, структура проекта, зоны интерфейса | ✅ Готово |
| 2 | Модели данных, карточка-образец, отрисовка | ✅ Готово |
| 3 | Конвейер, кнопки, State Machine | ⏳ В планах |
| 4 | Алгоритм крема — физика стекания | ⏳ В планах |
| 5 | Посыпка, многослойность | ⏳ В планах |
| 6 | Сравнение тортов, экран результата | ⏳ В планах |

## 🏗️ Структура проекта

```
CakeGame/
├── CakeGame/
│   ├── Models/
│   │   ├── CakeLayer.cs       # Модель одного слоя торта
│   │   └── Cake.cs            # Модель торта (список слоёв)
│   ├── Forms/
│   │   └── MainForm.cs        # Главное окно игры
│   ├── Rendering/
│   │   └── CakeRenderer.cs    # Отрисовка торта через GDI+
│   ├── Program.cs
│   └── CakeGame.csproj
├── docs/
│   └── plan.md                # Подробный план по этапам
└── README.md
```

## 🚀 Запуск

1. Открыть `CakeGame.sln` в Visual Studio 2022+
2. Убедиться, что выбран `.NET 6` или выше
3. Нажать `F5`

## 🎮 Управление (текущий этап)

- Кнопки на конвейере пока заглушки — появятся на Неделе 3
- На экране уже отображается случайный торт-образец (Неделя 2)

## 🛠️ Технологии

- C# 10 / .NET 6
- WinForms
